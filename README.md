# 🤖 Agentic AI Job Application System (LangGraph + LLM + Automation)

> 🚀 Built an Agentic AI system that improves job application success rate using LLM-based reasoning and personalization.

---

## 🚀 Overview

An end-to-end **Agentic AI system** that automates job discovery, intelligently ranks opportunities using LLM reasoning, and generates **tailored resume content** to maximize interview conversion.

Unlike traditional job portals, this system focuses on  **quality over quantity** , helping candidates apply smarter instead of applying blindly.

---

## 🎯 Key Features

* 🔍 Automated job scraping from Naukri using Playwright
* 🧠 LLM-based job ranking using Ollama (LLaMA3)
* 🎯 Personalized job scoring based on candidate profile
* 📄 Dynamic resume bullet generation per job
* ✉️ AI-generated cover messages
* 🔄 Multi-agent workflow using LangGraph
* 🖥 Streamlit dashboard for visualization
* ☁️ Deployed demo version for recruiters

---

## 🧩 System Architecture

```
Job Scraper (Playwright)
        ↓
Job Finder Agent
        ↓
LLM Ranking Agent (Ollama)
        ↓
Resume Optimization Agent
        ↓
Streamlit UI (Deployed Demo)
```

---

## ⚙️ Tech Stack

* **Agent Framework:** LangGraph
* **LLM:** Ollama (LLaMA3 - Local)
* **Automation:** Playwright
* **Backend:** Python
* **Frontend:** Streamlit
* **Data Handling:** JSON
* **Deployment:** Streamlit Cloud (Demo)
